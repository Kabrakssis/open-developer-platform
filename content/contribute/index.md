## Issues

If you are Symphony Customer or Partner and your issue relates to the Symphony platform and/or product, please refer to the [Symphony Support Portal](support.symphony.com).

If you're experiencing issues on a specific project hosted by the Foundation, please use the project issue tracker; for example, the Symphony Integrations Framework [issues are tracked on Github](https://github.com/symphonyoss/App-Integrations-Core/issues).

If you are requesting access to ODP, make sure to read the [Getting started page](getting-started/) before opening an issue.

For any other issue related with ODP access and functionalities, please [open a Github issue](https://github.com/symphonyoss/open-developer-platform/issues)

Below are reported issue tags that define which ODP component is affected.
1. **REST API** - Any issue related with the Symphony (Platform) REST API
2. **Embedding API** - Any issue related with the Symphony (Platform) Embedding API
3. **Extension API** - Any issue related with the Symphony (Platform) Extension API
